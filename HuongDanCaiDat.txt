================================================================================
1. YÊU CẦU HỆ THỐNG
================================================================================

  - Hệ điều hành: Windows 10/11, macOS, hoặc Linux (Ubuntu 20.04+)
  - RAM tối thiểu: 8 GB
  - Kết nối Internet ổn định (để tương tác với Google Cloud và các API)
  - Tài khoản Google Cloud Platform (GCP) với Billing đã được kích hoạt
  - Tài khoản Telegram để tạo Bot nhận cảnh báo

================================================================================
2. CÔNG CỤ CẦN CÀI ĐẶT
================================================================================

2.1. Terraform (>= 1.5)
  - Mục đích: Quản lý toàn bộ hạ tầng (Infrastructure as Code).
  - Tải về tại: https://developer.hashicorp.com/terraform/install
  - Cài đặt:
      + Windows: Tải file .zip, giải nén, thêm đường dẫn vào biến môi trường PATH.
      + macOS/Linux: sử dụng Homebrew hoặc package manager:
          brew install terraform
  - Kiểm tra: mở terminal, chạy lệnh:
      terraform --version

2.2. Google Cloud SDK (gcloud CLI)
  - Mục đích: Tương tác với GCP, xác thực tài khoản, quản lý tài nguyên.
  - Tải về tại: https://docs.cloud.google.com/sdk/docs/install-sdk
  - Cài đặt:
      + Windows: Tải file .exe và chạy trình cài đặt.
      + macOS/Linux: theo hướng dẫn trên trang Google.
  - Sau khi cài, chạy lệnh khởi tạo:
      gcloud init
      gcloud auth application-default login
  - Kiểm tra:
      gcloud --version

2.3. Python (>= 3.11)
  - Mục đích: Ngôn ngữ lập trình chính của 2 Cloud Function và script tấn công.
  - Tải về tại: https://www.python.org/downloads/
  - Lưu ý khi cài trên Windows: tick chọn "Add Python to PATH".
  - Kiểm tra:
      python --version
      pip --version

2.4. Git
  - Mục đích: Quản lý phiên bản mã nguồn.
  - Tải về tại: https://git-scm.com/downloads
  - Kiểm tra:
      git --version

2.5. gsutil (đi kèm Google Cloud SDK)
  - Mục đích: Mô phỏng tấn công bằng dòng lệnh CLI (gsutil cp).
  - Đã được cài sẵn khi cài Google Cloud SDK, không cần cài riêng.
  - Kiểm tra:
      gsutil version

  gcloud services enable generativelanguage.googleapis.com

6. CẤU HÌNH BIẾN MÔI TRƯỜNG
================================================================================

6.1. File terraform.tfvars
  - Sao chép file mẫu:
      copy terraform.tfvars.example terraform.tfvars      (Windows)
      cp terraform.tfvars.example terraform.tfvars         (macOS/Linux)
  - Mở file terraform.tfvars và điền các giá trị:
      + project_id            : ID dự án GCP
      + project_number        : Số dự án GCP
      + organization_id       : ID tổ chức GCP
      + region                : Vùng triển khai (mặc định: asia-southeast1)
      + gemini_api_key        : API key Gemini (lấy từ Google AI Studio)
      + tele_bot_token        : Token của Telegram Bot (lấy từ @BotFather)
      + tele_chat_id          : Chat ID của Telegram (lấy từ @userinfobot)

6.2. API Key bên ngoài
  - Gemini API Key: Truy cập https://aistudio.google.com/apikey để tạo.
  - OpenAI API Key (Fallback): Truy cập https://platform.openai.com/api-keys
    để tạo (chỉ cần thiết nếu muốn test cơ chế Dual-AI Fallback).
  - Telegram Bot: Tạo bot tại @BotFather trên Telegram, lấy Bot Token.
